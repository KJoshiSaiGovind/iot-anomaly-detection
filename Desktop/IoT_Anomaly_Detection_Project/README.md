# IoT Anomaly Detection Project

A comprehensive project for detecting anomalies in IoT network traffic using machine learning.

## Project Structure

- **data/**: Contains raw and processed datasets
- **models/**: Stores trained machine learning models
- **capture/**: Network packet capture utilities
- **preprocessing/**: Data preprocessing and feature extraction
- **training/**: Model training scripts
- **detection/**: Anomaly detection modules
- **webapp/**: Flask web application for visualization
- **report/**: Project reports and documentation
- **ppt/**: Presentation materials
